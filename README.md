## RD

RD é um desáfio de layout proposto em react.

### Demo

[Rd-frontend](https://rd-frontend.herokuapp.com/).

### Instalação

- renomeie o arquivo **.env-example** para **.env**
- `$ yarn install`

### Executar

#### 1. Build desenvolvimento

- `$ yarn start`

#### 2. Build Produção

- `$ yarn build`
- `$ yarn serve`
